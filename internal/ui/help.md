# stoptail

Elasticsearch TUI

## Global

| Key | Action |
|-----|--------|
| `Tab` / `Shift+Tab` | Switch views |
| `q` / `Ctrl+C` | Quit |
| `?` | Toggle help |
| `r` | Refresh data |

## Overview

| Key | Action |
|-----|--------|
| `/` | Focus filter |
| `Esc` | Clear all filters |
| `up/down/left/right` | Navigate grid |
| `Enter` | Open index in Workbench |
| `1-9` | Toggle alias filters |
| `U/R/I` | Show Unassigned/Relocating/Initializing |
| `c` | Create index |
| `d` | Delete selected index |
| `a` | Add alias to index |
| `A` | Remove alias from index |

## Workbench

| Key | Action |
|-----|--------|
| `Enter` | Activate editor |
| `Tab` | Trigger autocomplete / cycle focus |
| `Ctrl+R` | Execute request |
| `Ctrl+Y` | Copy body/response to clipboard |
| `Up/Down` | Navigate completions |
| `Esc` | Dismiss completions / deactivate |

## Mappings

| Key | Action |
|-----|--------|
| `/` | Filter indices |
| `Ctrl+F` | Search |
| `left/right` | Switch panes |
| `Enter` | Load mappings for selected index |
| `t` | Toggle tree/flat view |
| `up/down` | Scroll |

## Nodes

| Key | Action |
|-----|--------|
| `1/2/3` | Switch view (Mem/Disk/Fielddata) |
| `Ctrl+F` | Search |
| `up/down` | Scroll |
| `r` | Refresh |

## Tasks

| Key | Action |
|-----|--------|
| `Ctrl+F` | Search |
| `c` | Cancel selected task |
| `y/n` | Confirm/abort cancel |
| `up/down` | Select task |
| `r` | Refresh |

---

*Mouse supported - Press `?` to close*
